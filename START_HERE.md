## ❓ PERTANYAAN UMUM

**Q: Bagaimana jika saya mau ubah warna tema?**
A: Edit `style.css`, cari `:root { --primary-gold: #D4AF37 }`, ganti dengan warna lain

**Q: Foto tidak muncul?**
A: Pastikan foto ada di folder `image/`, nama file benar, format PNG/JPG

**Q: Map tidak muncul?**
A: Pastikan internet aktif, refresh halaman (F5)

**Q: Bagaimana cara print?**
A: Tekan Ctrl+P, design otomatis menyesuaikan untuk print

**Q: Bisa diakses offline?**
A: Ya! Setelah pertama kali dibuka, bisa tanpa internet (Service Worker)

**Q: Gimana cara share ke banyak orang?**
A: Buat URL berbeda untuk setiap orang: `index.html?to=NamaTamu`
