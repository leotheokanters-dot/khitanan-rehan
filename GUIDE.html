🚀 Quick Start

Untuk membuka undangan:
Buka file index.html di browser anda atau gunakan live server.

Website ini dirancang dengan tema islami yang elegan dengan warna putih, abu-abu, dan emas. Semua fitur sudah siap digunakan!

🎯 Personalisasi dengan URL Parameters

Anda dapat mempersonalisasi undangan dengan menambahkan parameter di URL. Ini sangat berguna untuk mengirimkan undangan ke individu yang berbeda.

Cara Penggunaan:

Format: index.html?to=NAMA_PENERIMA

Contoh URL:

📌 Untuk Pak Kartono:

index.html?to=Bapak%20Kartono

📌 Untuk Bu Tia:

index.html?to=Ibu%20Siti%20Atiahul%20Mubarokah

📌 Untuk Keluarga Besar:

index.html?to=Keluarga%20Besar%20Kami

📌 Untuk Khairul:

index.html?to=Khairul

📌 Untuk Bpk/Ibu Umum:

index.html?to=Bapak/Ibu%20Saudara/i

⚠️ Tips: Spasi dalam URL harus diganti dengan %20. Misalnya "Bapak Kartono" menjadi "Bapak%20Kartono".

🎨 Cara Kustomisasi

1. Mengubah Nama Anak

Buka file index.html dan cari:

<h2 class="child-name">Muhammad Rehan</h2>

Ubah "Muhammad Rehan" dengan nama anak Anda.

2. Mengubah Tanggal Acara

Cari di index.html:

Minggu, 12 April 2026
24 Syawal 1447 H
Jam 09:00 WIB

Dan di script.js:

const targetDate = new Date('2026-04-12T09:00:00')

3. Mengubah Warna Tema

Buka file style.css dan ubah root variables:

:root {
--primary-gold: #D4AF37; /* Gold utama */
--primary-gold-light: #E8C547; /* Gold terang */
--white: #FFFFFF; /* Putih */
--text-dark: #2C2C2C; /* Teks hitam */
}

4. Mengganti Foto

Foto berada di folder image/:

rehan1.png → Foto di hero section
rehan2.png → Foto di halaman utama
rehan3.png, rehan4.png, rehan5.png → Galeri
keluarga1.png, keluarga2.png → Foto keluarga

Anda bisa mengganti file foto dengan nama file yang sama, atau mengubah nama file di HTML.

🔧 Troubleshooting

Map tidak muncul

Pastikan koneksi internet aktif
Refresh halaman (tekan F5)
Cek console browser untuk error (tekan F12)

Foto tidak muncul

Pastikan foto ada di folder image/
Periksa nama file foto (case-sensitive)
Format harus PNG atau JPG

URL parameter tidak bekerja

Pastikan URL format benar: index.html?to=NAMA
Ganti spasi dengan %20
Refresh halaman setelah mengubah URL

Countdown tidak update

Clear browser cache (Ctrl+Shift+Delete)
Refresh halaman
Pastikan jam sistem komputer sudah benar
